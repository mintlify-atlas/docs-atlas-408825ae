illogical-impulse ships a comprehensive set of keybinds organized across the shell, window management, workspaces, utilities, media, and app launching. Press `Super+/` at any time to open the interactive in-shell cheatsheet.

<Tip>
  Add your own keybinds in `~/.config/hypr/custom/keybinds.conf`. This file is sourced after the defaults and will not be overwritten by updates.
</Tip>

## Shell

| Keybind | Action |
|---------|--------|
| `Super` (hold/release) | Toggle search / launcher |
| `Super+Tab` | Toggle workspace overview |
| `Super+A` | Toggle left sidebar (AI / Translator) |
| `Super+N` | Toggle right sidebar (notifications / controls) |
| `Super+/` | Toggle keybind cheatsheet |
| `Super+M` | Toggle media controls |
| `Super+J` | Toggle bar |
| `Super+K` | Toggle on-screen keyboard |
| `Super+G` | Toggle overlay |
| `Super+V` | Clipboard history → clipboard |
| `Super+Period` | Emoji picker → clipboard |
| `Ctrl+Super+T` | Open wallpaper selector |
| `Ctrl+Super+Alt+T` | Select random wallpaper |
| `Ctrl+Super+R` | Restart shell widgets |
| `Ctrl+Super+P` | Cycle panel family (ii ↔ Waffle) |
| `Ctrl+Alt+Delete` | Toggle session menu |

## App launchers

| Keybind | Action |
|---------|--------|
| `Super+Enter` or `Super+T` | Terminal |
| `Ctrl+Alt+T` | Terminal (Ubuntu-style alias) |
| `Super+E` | File manager |
| `Super+W` | Browser |
| `Super+C` | Code editor |
| `Super+X` | Text editor |
| `Super+I` | Settings app |
| `Ctrl+Super+V` | Volume mixer |
| `Ctrl+Shift+Escape` | Task manager |

These app variables are set in `~/.config/hypr/hyprland/variables.conf` and can be overridden in `~/.config/hypr/custom/variables.conf`:

```ini
$terminal    = kitty -1
$fileManager = dolphin
$browser     = firefox
$codeEditor  = code
```

## Virtual machine mode

Press `Super+Alt+F1` to enter the virtual machine submap — all Hyprland keybinds are disabled so you can interact with the VM freely. Press `Super+Alt+F1` again to re-enable them.

## Customizing keybinds

Add your own binds to `~/.config/hypr/custom/keybinds.conf`:

```ini custom/keybinds.conf
# Example: open a browser with Super+B
bind = Super, B, exec, firefox

# Example: launch a scratchpad terminal
bind = Super+Alt, T, exec, kitty --class scratchpad
```

This file is never overwritten by updates.
