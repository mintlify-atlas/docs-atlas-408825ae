## What is illogical-impulse?

illogical-impulse is technically a collection of configuration files, but realistically it is a complete, opinionated desktop environment built on top of Hyprland. The centrepiece is a custom shell — written in QML and powered by Quickshell — that handles everything a traditional desktop environment panel would: workspace switching, system tray, notifications, media controls, brightness/volume, AI chat, and more.

<Steps>
  <Step title="Install the dotfiles">
    Run the one-liner to clone the repo and launch the transparent setup script. Every command is shown before it executes.

    ```bash
    bash <(curl -s https://ii.clsty.link/get)
    ```
  </Step>
  <Step title="Launch Hyprland">
    Select the **Hyprland** session from your display manager, or start it from a TTY. The shell loads automatically on login.
  </Step>
  <Step title="Pick a wallpaper">
    Press `Ctrl+Super+T` to open the wallpaper selector. Choose any image and Material Design 3 colors are generated automatically — the entire shell recolors instantly.
  </Step>
  <Step title="Explore the shell">
    Press `Super+/` to view the keybind cheatsheet. Press `Super+A` for the AI sidebar, `Super+N` for notifications and quick toggles, and `Super` to open the search launcher.
  </Step>
</Steps>

## Key features

<CardGroup cols={2}>
  <Card title="Material Design 3 theming" icon="swatchbook">
    Every color in the shell — bar, sidebars, notifications, lock screen — is derived from your wallpaper automatically via matugen.
  </Card>
  <Card title="AI sidebar" icon="brain">
    A built-in chat interface supports Gemini, any OpenAI-compatible endpoint, and local Ollama models. Attach files, save chats, and switch models on the fly.
  </Card>
  <Card title="Screen utilities" icon="scissors">
    OCR to clipboard, Google Lens image search, screen translation, color picker, region screenshot, and screen recording — all bound to easy keybinds.
  </Card>
  <Card title="Workspace overview" icon="table-cells">
    Press Super+Tab for a live-preview workspace switcher showing all open windows across all workspaces.
  </Card>
  <Card title="Transparent installer" icon="shield-check">
    The setup script shows every command before running it, is fully idempotent, and supports Arch, Fedora, Gentoo, and NixOS.
  </Card>
  <Card title="Waffle panel family" icon="microsoft">
    Switch to the Windows 11-inspired Waffle layout with Ctrl+Super+P for a taskbar-style experience with an Action Center.
  </Card>
</CardGroup>

<Note>
  illogical-impulse is a Hyprland-only project. It requires a Wayland compositor and will not work on X11. If you are new to Linux or Hyprland, read the [introduction](/introduction) before installing.
</Note>
