illogical-impulse is a collection of configuration files that, in practice, delivers a complete graphical shell for the Hyprland Wayland compositor. At its core is a custom widget system written in QML and powered by Quickshell — covering the status bar, sidebars, workspace overview, search launcher, notifications, and more. Colors across the entire shell are generated automatically from your wallpaper using Material Design 3 color generation via matugen.

<Warning>
  illogical-impulse is **not** a system setup script. It does not install GPU drivers, configure zram, tune swap, or perform any hardware-level setup. You are expected to have a working Arch, Fedora, Gentoo, or NixOS installation with a functional GPU driver before running the installer.
</Warning>

## What it is

Technically, illogical-impulse is a set of configuration files. In practice, it is mostly a custom graphical shell that sits on top of Hyprland. The installer handles package installation, user group configuration, service setup, and copying config files into your home directory — but it stops there.

The installer is also transparent: every command is shown on screen before it executes, and you confirm each step. The process is idempotent, so you can safely re-run it to apply updates or restore a misconfigured state.

## What it is not

- **Not a system setup script.** No GPU drivers, no zram, no swap configuration, no display manager setup.
- **Not a beginner hand-holder.** The project itself notes that if you are new to Linux and choose Hyprland, you are in for a tough ride.
- **Not cross-compositor.** illogical-impulse is Wayland-only and built specifically for Hyprland. It will not run on X11 or any other compositor.

## Software stack

The shell is built from several upstream projects working together:

| Software | Purpose |
|---|---|
| [Hyprland](https://github.com/hyprwm/hyprland) | Wayland compositor — manages and renders all windows |
| [Quickshell](https://quickshell.outfoxxed.me/) | QtQuick-based widget system powering the bar, sidebars, overview, and more |
| matugen | Generates a Material Design 3 color palette from your wallpaper |
| Fish | Default shell, with Starship prompt and eza-based `ls` |
| Kitty | Default terminal emulator |
| Fuzzel | Application launcher (also used as a search bar fallback) |
| wlogout | Session menu (lock, sleep, shutdown, reboot) |
| hyprlock / hypridle | Lock screen and idle daemon |

A full list of all packages and their roles is maintained in the repository at `sdata/deps-info.md`.

## Supported distributions

Arch Linux is the primary supported distribution and is actively maintained by the core developers. Fedora, Gentoo, and NixOS are supported by the community — their dependency scripts may occasionally lag behind the Arch reference. If you are on a community-supported distribution, check the upstream [discussions board](https://github.com/end-4/dots-hyprland/discussions/2140) before installing.

| Distribution | Status |
|---|---|
| Arch Linux | Primary — actively maintained by core developers |
| Fedora | Community-supported |
| Gentoo | Community-supported |
| NixOS | Community-supported (experimental, work in progress) |
